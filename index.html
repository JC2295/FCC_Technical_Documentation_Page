Introduction

The purpose of this web page is to provide docummentation fot setting up and utilising the C++ programming language. This documentation is primarily cover the installlation and setup of a chosen Integrated Developemnet Environment (IDE). Following this the basic functionality and syntax of the C++ programming language will be covered with example code snippets and reference material which will be linked to for further information.

Prerequisites

This documenation has been designed for intermediate users. Therefore it is assumed that all users who utilse this documentation are computer literate and pocess basic knowledge of computer systems and programming methodology. Those who do not possess this knowledge should follow the Free Code Camp origination to learn more.

Programming

C++ is a general purpose programming language developed in 1983. It was developed from the C programming language developed in the 1970's which was used as a foundation to build it on. C++ was designed to work very well with hardware whilst also being able to handle the complexity that is required to work with real world appliactions. Even to this very day the're few languages that meet this criteria in this way which is why you can still C++ code being utilised in a applications. These include game development, simulation and robotics whith a a notable example being the C++ code utilised in the mars rover. It is a highly practical language and well worth learning and it is hoped that this documentation will assist with that.

Setup

C++ is a compiled programming language which means it requires what is known as a compiler. A compiler is a piece of software which converts the C++ code that has been written into machine code so that the machine can understand and run the program. Without this we cannot write any code. So before we can begin we must install a compiler program onto the computer

Comments

Comments are a verey important part of programming. They are sets of text that the program ignores and are used to give context to what the program does in order to aid devlopers when writing software. There are two different types of syntax used in C++ to leave comments. These are shown below:

  • // Single line comment.
  • /*Multi - line comment.*/
